# trabajo-final

## ¿Para qué es esta propuesta?
##### Esta propuesta tiene como proposito de ser mi portafolio que contiene los trabajos que ya he realizado adentro y afruera de mis estudios de communicación visual. Además ofrece la opción de contactarme y encontrar más canales de mi contenido. #####

#### Demografía: Un productor audio-visual que tiene un equipo grande que hace proyectos que me interesan. El esta buscando nuevos talentos para sus próximos proyectos.
